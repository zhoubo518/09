二、第一节 应急管理法规与政策的主要构成

1. 突发事件与应急管理的定义

1.1 突发事件

①定义：突然发生，造成或可能造成严重社会危害，需要采取应急处置措施予以应对的自然灾害、事故灾难、公共卫生事件和社会安全事件。

②四个基本特征：

  • 突发性或公共性
  • 突发性和紧迫性
  • 危害性和破坏性
  • 必须以权力组织应对

③四类分级：

按社会危害程度、影响范围等因素，分为特别重大、重大、较大和一般四级。

1.2 应急管理

原则："预防为主、预防与应急相结合"

①管理体系：在党委领导、政府负责、社会协同、公众参与、法治保障的框架下，以政府作为管理主体，全社会共同参与。

管理体制：

  • 统一领导、专常兼备、反应灵敏、上下联动
  • 综合协调、分类管理、分级负责、属地管理为主

②五个核心特征：

特征内容
公共性应急管理的主体、对象和过程都在公共治理框架内
系统性目标、对象、流程由多个要素汇聚、相互作用形成体系
前瞻性掌握突发事件规律，开展预警和预防，提前组织干预
专业性由专业人员、知识、技术应对不同领域的突发事件
常态性不仅包含应急状态下的处置，也涵盖常态下的预防、减灾、备灾等工作

速记口诀：「公系前专常」——公共性、系统性、前瞻性、专业性、常态性

2. 应急管理的目的——五个目的

  1. 保护人民生命财产安全——以人民为中心是根本宗旨，最大程度保护人民生命财产安全。【人安】
  2. 维护国家安全、公共安全、生态环境安全和社会秩序——外在目标是安全，内在要求是结果导向；从公共性来看，外在目标是公平，内在要求是过程导向。【国安】
  3. 预防和减少突发事件的发生，控制、减轻和消除突发事件引起的严重社会危害——坚持预防为主、源头治理，通过事前、事中、事后的连续性管理，实现预防与应急相结合。【预防控制】
  4. 提高突发事件预防和应对能力——通过预防与准备、监测与预警、处置与救援、恢复与重建、调查评估与总结改进等环节，防范重大风险，提升应急能力。【提能力】
  5. 规范突发事件应对活动——防止发生次生或衍生灾害。【规范应对】

核心目的：保护人民生命财产安全、维护国家安全与社会秩序

速记口诀：「人安、国安、预防控制、提能力、规范应对」

3. 应急管理的功能——五大功能

功能说明
1. 保护人民群众生命财产安全核心功能
2. 推进国家治理体系和治理能力建设推进国家治理
3. 提升安全服务水平高质量发展将安全贯穿经济社会发展各方面
4. 防范重大安全风险和及时处置各类灾害事故不断提升防范和处置能力
5. 维护社会稳定筑牢防灾减灾的人民防线

速记口诀：「保护生命财产安全、推进国家治理、提升安全服务、防范风险与处置灾害、维护社会稳定」

4. 应急管理的法规政策体系

1. 2019年党的十九届四中全会明确提出——"构建统一指挥、专常兼备、反应灵敏、上下联动的应急管理体制，优化国家应急管理能力体系建设，提高防灾减灾救灾能力"。

2. 形成"1+4"的应急预案体系："1"指国家突发事件总体应急预案；"4"指覆盖自然灾害、事故灾难、公共卫生事件、社会安全事件4类突发事件的专项应急预案。

三、第二节 应急管理法规与政策的有关要求

一、《突发事件应对法》（2024年修订）核心制度框架

1. 管理与指挥体制

中央层面：国务院统一领导全国应急工作，设立国家突发事件应急指挥机构，必要时派出工作组。

县级以上层面：政府主要负责人、相关部门负责人、救援队伍及驻地部队负责人组成应急指挥机构，统一领导、协调本区域应对工作。【县级以上——指导、领导、协调】

县级层面：对本区域突发事件应对负总责，事件发生后立即处置并向上一级政府报告，必要时可越级上报。【县级—立即干】

乡级与街道层面：明确专门工作力量负责应急工作。【乡镇街道—专门】

基层自治组织：村（居）委会协助政府开展工作，公民、法人有义务参与应对。【基层组织—协助】

速记口诀：「中央统一领导，县级以上指导协调，县级负总责立即干，乡镇专门力量，基层协助」

2. 县级及县级以上人民政府应对机制

①报告机制：突发事件发生后要立即向上一级人民政府报告，而且规定必要时可以越级上报。如果遇到不能消除或者不能有效控制的严重危害社会的事件应当及时向上级人民政府报告。具备条件的，可以网络直报或自动速报。【报告上一级——县】

②跨区域应对：突发事件涉及两个以上行政区域的，其应对管理工作由有关行政区域共同的上一级人民政府负责，或者由各有关行政区域的上一级人民政府共同负责。【报告—共同上一级、上一级共同】

③应急指挥机构：如果实际有需要，可以设立与突发事件类别有关的应急指挥机构。需要注意的是，突发事件应急指挥机构发布的决定、命令、措施仅限于"突发事件应对过程中"。

④人大监督：应急处置结束后，县级以上政府需向本级人大常委会作专项工作报告。【县级以上—本级人大】

二、突发事件应对过程全阶段要求

1. 预防与准备阶段

（1）应急预案体系

  • 国务院制定国家总体预案与专项预案，各部门、地方政府制定对应预案，形成"国家 - 地方 - 基层"三级体系。
  • 预案需根据实际情况动态修订，应急管理部门负责衔接与指导。

速记口诀：「国家 - 地方 - 基层三级预案，动态修订」

（2）风险防控与基础设施

  • 县级以上政府将应急工作纳入国民经济规划，统筹建设应急避难场所、医疗救治点等设施。【基础设施—县+政府】
  • 所有单位需建立安全管理制度，定期开展隐患排查与风险评估。

（3）加强风险评估体系和危险源、危险区域的管控

县级人民政府：对本行政区域内一般、较大突发事件的危险源、危险区域进行调查、登记、风险评估，定期检查、监控，责令有关单位采取安全防范措施。【一般、较大——调查登记评估】

县级以上地方人民政府：根据情况变化，及时调整危险源、危险区域的登记。【县+政府——确定、调整区域】

省级和设区的市级人民政府：对本行政区域内容易引发特别重大、重大突发事件的危险源、危险区域进行调查、登记、风险评估，组织检查、监控，责令采取安全防范措施。【重大、特大——市级省级——调查登记评估】

所有单位：建立健全安全管理制度，定期开展危险源辨识评估。

危险源、危险区域及其基础信息——接入突发事件信息系统，登记、管控、及时公布。

速记口诀：「一般较大归县级，重大特大归市省级」

（4）应急队伍与保障

①培训制度

县级以上人民政府——建立健全突发事件应对管理培训制度，对负有突发事件应对管理职责的工作人员以及居民委员会、村民委员会有关人员定期进行培训。

②协同

  • 国家综合性消防救援队伍——综合性常备骨干力量
  • 县级以上人民政府及其有关部门——可以建立由成年志愿者组成的应急救援队伍
  • 乡级人民政府、街道办事处和有条件的居民委员会、村民委员会——可以建立基层应急救援队伍
  • 单位——应当建立由本单位职工组成的专职或兼职应急救援队伍
  • 专业应急救援人员应取得国家规定的应急救援职业资格。

③夯实应急救援的社会力量

  • 加强社会公众的应急知识宣传和必要的应急演练；
  • 教育主管部门应当对学校开展应急教育进行指导和监督，各级各类学校要对学生及教职工开展应急知识教育和应急演练；
  • 鼓励公民、法人和其他组织储备基本的应急自救物资和生活必需品；
  • 鼓励社会力量参与突发事件应急运输保障；
  • 鼓励为突发事件应对工作提供物资、资金、技术支持和捐赠。

2. 监测与预警

（1）建立全国统一的信息系统

  • 县级以上人民政府有关主管部门应当向本级人民政府相关部门、上级人民政府主管部门通报突发事件信息。
  • 专业机构、监测网点和信息报告员应当及时向所在地人民政府及其主管部门报告突发事件信息。
  • 县级以上地方人民政府应当及时汇总分析突发事件隐患和监测信息，必要时组织会商，进行突发事件影响评估。【评估——县+政府】
  • 评估结果认为可能发生重大或特别重大突发事件的，县级以上地方人民政府应当立即向上级人民政府报告，并向毗邻或相关地区的人民政府通报，及时采取预防措施。【重特大——县+政府报上级政府】

（2）建立预警制度

①预警制度与预警范围、分级

  • 范围：主要包括自然灾害、事故灾难、公共卫生事件
  • 危害程度：一级（红）、二级（橙）、三级（黄）、四级（蓝）
  • 划分标准制定：国务院或国务院确定的部门

②发布前提：事件即将发生或者发生的可能性增大

③发布主体：县级以上地方人民政府

注意区分：预警级别分为一级、二级、三级、四级（红橙黄蓝）；突发事件等级分为特别重大、重大、较大、一般。两者不要混淆！

④程序

发布相应级别的警报 → 决定并宣布相关地区进入预警期，同时向上一级人民政府报告，必要时可以越级上报，同时向当地驻军和可能受到危害的毗邻或相关地区的人民政府通报。具备条件的，应当进行网络直报或者自动速报。

(4) 定时向社会发布评估结果
(5) 及时发布可能受到突发事件危害的警告，宣传避免、减轻危害的常识，公布联络方式和渠道

速记口诀：「一、二级预警需紧急疏散、管控风险；三、四级预警需启动预案、发布警示」

⑥预警发布后的保障措施

市场监测和保障供应——对重要商品和服务市场情况加强监测；根据实际需要及时保障供应、稳定市场。

⑦预警级别调整和解除

有事实证明不可能发生突发事件或者危险已经解除的，发布警报的人民政府应当立即宣布解除警报，终止预警期，并解除已经采取的有关措施。

3. 应急处置与救援

（1）应急响应级别的划分、划分标准和确定

  • 级别：一级、二级、三级、四级。一级为最高级别。
  • 制定级别划分标准：国务院或者国务院确定的部门
  • 确定应急响应级别：县级以上人民政府及其有关部门

（2）应急响应的领导职责、组织处置和协调

①启动应急响应并采取处置措施的主体——人民政府——包括国务院在内的县级以上的人民政府。

②处置措施：

  • 自然灾害 / 事故灾难 / 公共卫生事件：营救人员、疏散安置、医疗救治、管控风险、保障生活
  • 社会安全事件：强制隔离、管制现场、保卫重点单位

③报告制度和协调机制：

  • 启动应急响应需要向上级人民政府报告，并建立协调机制。
  • 在必要时，可以设立现场指挥部负责现场应急处置与救援，做统一指挥。
  • 引导社会力量及时有序参与应急处置与救援工作。

（3）应急征用、特殊群体帮扶与社会力量参与

  • 必要时可以向有关单位和个人征用应急救援所需设备、设施、场地、交通工具和其他物资，并在运送方面有优先性。
  • 应当为受突发事件影响无人照料的无民事行为能力人、限制民事行为能力人提供及时有效帮助。

调动社会力量参与应急处置与救援：

  • ①个人义务：服从指挥与安排，配合采取应急处置措施，积极参与应急救援工作，协作维护社会秩序。
  • ②城乡社区组织义务：加强与突发事件信息系统数据共享，增强保障群众基本生活和服务群众的能力。
  • ③心理服务、社会工作与人才队伍建设：支持引导心理健康服务人员和社会工作者对受突发事件影响的各类人群开展心理援助工作。

（4）个人信息与个人尊严保护

①个人信息提供义务与个人信息严格保密原则：县级以上人民政府及其有关部门对获取的相关信息，应当严格保密，并依法保护公民的通信自由和通信秘密。

②信息安全保护和信息获得的合法性：

  • 不得非法收集、使用、加工、传输他人的个人信息，不得非法买卖、提供或者公开他人的个人信息。
  • 只能用于突发事件应对，并在突发事件应对工作结束后予以销毁。

4. 事后恢复与重建

（1）解除和恢复

  1. 宣布解除应急响应，停止已采取的应急处置措施
  2. 采取或继续采取实施必要的措施，防止自然灾害、事故灾难、公共卫生事件的次生、衍生事件或者重新引发社会安全事件
  3. 立即组织调查评估，制定恢复重建计划，并向上一级人民政府报告。

（2）恢复与重建的政策支持

  • ①寻求上一级系统支持：需要上一级人民政府支持的，可以向上一级人民政府提出请求。
  • ②制定和落实优惠与补助政策：
    • 志愿者、公民——参加应急救援工作或者协助维护社会秩序期间，其所在单位应当保证其工资待遇和福利不变，并可以按照规定给予相应补助。
    • 受伤人员——县级以上人民政府对在应急救援工作中伤亡的人员依法落实工伤待遇、抚恤或者其他保障政策，并组织做好应急救援工作中致病人员的医疗救治工作。
  • ③制定善后工作计划，解决矛盾纠纷
  • ④总结经验教训，加强审计与档案移交：
    • 查明经过和原因，总结处置工作的经验教训，制定改进措施，并向上一级人民政府提出报告。
    • 有关资金、物资的筹集、管理、分配、拨付和使用等情况，应当依法接受审计机关的审计监督。
    • 突发事件应对工作中形成的材料，应当按照国家规定归档，并向相关档案馆移交。

三、《突发事件应急预案管理办法》

1. 管理体制

国务院统一领导，县级以上地方政府负责本区域预案管理，应急管理部门综合协调。

2. 应急预案分类与内容

（1）分类

  • 政府及部门应急预案：总体预案、专项预案、部门预案。
  • 单位和基层组织应急预案：企业、村（居）委会、社会组织等预案。

（2）核心内容

①总体预案：应对突发事件的总体制度安排，明确工作的总体要求、事件分类分级、预案体系构成、组织指挥体系与职责，以及风险防控、监测预警、处置救援、应急保障、恢复重建、预案管理等。

②专项应急预案：应对某一类型或某几种类型突发事件，或者重要专项工作而预先制定的涉及多个部门职责的方案。

③部门应急预案：应对本部门（行业、领域）突发事件，或针对涉及部门工作而预先制定的方案。

④基层预案：乡镇（街道）、村（社区）等基层组织的预案，突出实用性与灵活性。

速记口诀：「总体预案为制度安排，专项预案针对特定事件，部门预案按职责制定，基层预案突出实用性」

3. 应急预案制修订工作计划

  • 国务院：应急管理部门会同有关部门编制应急预案制修订工作计划，并报国务院批准后实施。
  • 本行政区域：县级以上地方人民政府应急管理部门应当会同有关部门，针对本行政区域多发易发突发事件、主要风险等，编制本行政区域应急预案制修订工作计划，报本级人民政府批准后实施，并抄送上一级人民政府应急管理部门。
  • 本部门：县级以上人民政府有关部门可以结合实际制定本部门（行业、领域）应急预案编制计划，并抄送同级应急管理部门，同时抄送上一级相应部门。

4. 预案编制小组

应急预案编制部门、单位、有关部门和单位人员、有关专家、有应急处置工作经验的人员。

小组组长——应急预案编制部门或单位有关负责人担任。

5. 预案的审批与发布

①国家层面：

  • 国家总体预案：党中央、国务院审批发布
  • 专项应急预案：由预案编制牵头部门送应急管理部衔接协调后，报国务院审批
  • 部门应急预案：由部门会议审议决定

②地方层面：

  • 地方总体预案：本级党委和政府审批发布
  • 专项应急预案：送本级应急管理部门衔接协调，报本级人民政府审批
  • 部门应急预案：按照本级人民政府和上级有关部门的应急预案管理规定执行

③其他：

  • 重大活动保障应急预案、巨灾应急预案：由本级人民政府或其部门审批
  • 跨行政区域联合应急预案：由相关人民政府或其授权的部门协商确定审批

④单位 / 基层预案：本单位 / 组织负责人签发发布

速记口诀：「国家总体预案由党中央、国务院发布；地方预案由本级党委和政府发布」

6. 预案的备案与演练

  • 预案发布后20个工作日内报送备案；
  • 专项预案、部门预案每3年至少演练1次；
  • 人员密集场所的经营单位或者管理单位等，应当有针对性地组织开展应急预案演练。

7. 预案的评估与修订

评估：应急预案编制单位应当建立应急预案定期评估制度。

及时修订预案的情形：

  • 有关法律法规、规章、标准发生重大变化；
  • 应急指挥机构及其职责发生重大调整；
  • 面临的风险发生重大变化；
  • 重要应急资源发生重大变化；
  • 在突发事件实际应对和应急演练中发现问题需要作出重大调整。

四、本章总结

突发事件基本框架

突发事件：4类（自然灾害、事故灾难、公共卫生事件、社会安全事件），4级（特别重大、重大、较大、一般）

管理与指挥体制

  • 中央层面：国务院统一领导，设立国家突发事件应急指挥机构
  • 县级以上层面：政府主要负责人牵头，组成应急指挥机构
  • 县级层面：本区域应对工作负总责，事件发生后立即处置并上报，必要时可越级上报
  • 乡级与街道：明确专门工作力量；村（居）委会协助工作
  • 人大监督：处置结束后向本级人大常委会作专项工作报告

预防与应急准备

  • 预案体系：国家 - 地方 - 基层三级预案，动态修订
  • 基础设施：国土空间规划需满足应急需求，建设应急避难场所
  • 风险防控：一般、较大——县级调查登记评估；重大、特大——市级省级调查登记评估；县+政府——确定、调整区域
  • 队伍建设：国家综合性消防救援队伍为骨干，专业、基层、社会应急力量协同
  • 社会动员：应急知识进学校，村（居）委会、企业定期演练
  • 保障体系：物资、运输、能源、通信、医疗等保障，科技支撑

监测与预警

  • 信息报告：及时、客观、真实，公民与单位有报告义务
  • 预警分级：一级（红）、二级（橙）、三级（黄）、四级（蓝）
  • 预警发布：县级以上地方人民政府发布，明确级别、范围与措施
  • 预警响应：一、二级预警需紧急疏散、管控风险；三、四级预警需启动预案、发布警示

应急处置与救援

  • 响应分级：一级至四级，与事件、预警分级对应
  • 处置主体：统一领导或组织处置的人民政府
  • 处置措施：自然灾害/事故灾难/公共卫生事件——营救人员、疏散安置、医疗救治、管控风险、保障生活；社会安全事件——强制隔离、管制现场、保卫重点单位
  • 特殊群体帮扶：保障无民事/限制民事行为能力人等群体权益
  • 社会力量参与：鼓励企业、社会组织、志愿者参与救援

事后恢复与重建

  • 终止应急：危害消除后解除响应，防范次生事件
  • 重建规划：损失评估、制定规划、争取上级支持
  • 恢复保障：救助、补偿、抚慰受灾人员，恢复基础设施
  • 总结评估：复盘工作，完善预案，接受审计监督
